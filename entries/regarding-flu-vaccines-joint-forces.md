# Regarding the end of the mandatory flu vaccine requirement in the US Joint Forces

Let the sick die, so only the healthy survive.  
Let the starving perish, so only the fed survive.  
Let the peaceful fail, so only warriors survive.  
Then when there is no medicine, no food, and no kindness  
surely our healthy fed warriors  
will want for nothing.
